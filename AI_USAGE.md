# AI Usage Notes

## Which AI tools were used?

- OpenAI Codex / GPT-5 coding agent
- Stitch MCP for UI design direction and reference screens

## How were they used during the work?

- The coding agent was used to read the assignment brief, plan the implementation, scaffold the app, write the React and TypeScript code, and add automated tests.
- Stitch was used to generate a design basis for the two required screens so the UI had a clear visual direction before implementation.
- AI was also used to draft the delivery documentation and verification flow.

## What would I improve with more time?

- Add edit and delete actions for tasks
- Improve empty-state and success-state feedback even more
- Add optional filters such as completed and incomplete
- Refine the visual system further with richer animations and a custom favicon/icon set
